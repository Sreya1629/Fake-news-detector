# 📰 Fake News Detection System

## 📌 Overview
This project detects whether a news article is Fake or Real using Machine Learning and Flask web application.

---

## 🚀 Features
- Fake / Real news classification
- Confidence score prediction
- Simple and interactive web interface
- Real-time prediction using Flask

---

## 🧠 Tech Stack
- Python
- Machine Learning (Scikit-learn)
- TF-IDF Vectorization
- Logistic Regression
- Flask
- HTML & CSS

---

## 📂 Project Structure
- train.py → Model training
- app.py → Web application
- dataset/ → Dataset files
- templates/ → HTML files
- static/ → CSS files

---

## ⚙️ How to Run

1. Install dependencies:
    pip install flask pandas scikit-learn joblib 
2. Train model:
    python train.py
3. Run application:
    python app.py
4. Open browser:
    http://127.0.0.1:5000


---

## 📊 Output
User enters news → System predicts Fake or Real with confidence score.
